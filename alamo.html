Website Development for
Alamo Conservation Project


This project is to design and develop a website for a conservator of Alamo to input the information of the wall through just several clicks, the visualize the recording process to make it easier and quicker to record information.

Homepage



Log in page



Map page

This map is a manuscript for Alamo historic site, the letter from A to R stands for walls, and number stands for pillars. All the green and red triangles are clickable. However, since there are only picture for WallA, we made WallA green and others red.



Walls page

This is a second way to go to one specific wall, instead of finding a wall on the map, we can also use this more direct way to go to single wall page.



The Single Wall page

For Visitors

Visitors can only see the information that have been added to the all, they cannot change any of them.

For Administrator

This page is for administrator, the administrator can add information to this wall, and he can view the information he already added as well.
